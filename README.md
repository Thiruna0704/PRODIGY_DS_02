🚢 Titanic Dataset – Data Cleaning & Exploratory Data Analysis (EDA)
📌 Project Overview

This project focuses on performing Data Cleaning and Exploratory Data Analysis (EDA) on the famous Titanic Dataset.
The goal is to analyze relationships between variables and identify patterns that influenced passenger survival.

📂 Dataset Information

The dataset contains passenger details such as:

PassengerId

Pclass (Passenger Class)

Name

Sex

Age

SibSp (Siblings/Spouses aboard)

Parch (Parents/Children aboard)

Ticket

Fare

Cabin

Embarked

Survived (Target Variable)

Dataset Source: Kaggle – Titanic Dataset

🛠️ Technologies Used

Python 🐍

Pandas

NumPy

Matplotlib

Seaborn

Google Colab / Jupyter Notebook

🧹 Data Cleaning Steps

Handled missing values in Age, Embarked, and Fare

Filled missing values using:

Median (Age, Fare)

Mode (Embarked)

Dropped Cabin column due to high missing data

Checked and verified no remaining null values

📊 Exploratory Data Analysis (EDA)

The following analysis was performed:

Survival count visualization

Survival rate by Gender

Survival rate by Passenger Class

Age distribution

Fare vs Survival analysis

Correlation heatmap

🔍 Key Insights

👩 Female passengers had a higher survival rate than males

🏆 1st class passengers survived more compared to 2nd & 3rd class

💰 Higher fare passengers had better survival chances

👶 Younger passengers showed slightly better survival trends

🎟️ Passenger class and fare are strongly correlated
